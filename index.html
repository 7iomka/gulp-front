Text level elements

Text level HTML elements may be used within other elements. They include:

  • <a href="#">link or anchor</a>
  • <abbr> represents an abbreviation or acronym
  • <em> for emphasized text
  • <strong> for strongly emphasized
  • <dfn> for definition of a term
  • <code> for code fragment
  • <samp> for sample text
  • <kbd> for keyboard input
  • <var> for variable
  • <cite> for short citation
  • <i> for italics
  • <b> for bold
  • <u> for underlined text
  • <s>for text that is no longer correct
  • <big> for larger text
  • <small> for smaller text
  • <sup> for superscript
  • <sub> for subscript
  • <del> for visibly deleted
  • <ins> for inserted content
  • <dfn> for definitions
  • <mark> for highlighted passages
  • <hr> for horizontal line

Notifications

Options

class

A string containing one or more extra CSS classes

  • Type: string
  • Default: ''

click

Click callback

  • Type: function
  • Default: none

content

Notification content

  • Type: string
  • Default: ''

duration

On screen duration in milliseconds, set it to 0 for sticky

  • Type: integer
  • Default: 5000

fadeIn

Fade in effect duration in milliseconds

  • Type: integer
  • Default: 400

fadeOut

Fade out effect duration in milliseconds

  • Type: integer
  • Default: 400

limit

Visible limit per board, when reached notifications wont be created

  • Type: integer, boolean
  • Default: false

queue

If queue is active and limit is reached notifications will be shown when other hides

  • Type: integer, boolean
  • Default: false

slideUp

Slide up effect duration in milliseconds

  • Type: integer
  • Default: 200

horizontal

Horizontal alignment, default right

  • Type: string
  • Default: ''

vertical

Vertical alignment, default top

  • Type: string
  • Default: ''
